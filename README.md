# Calendar App

---

## Functionality  
App allows managing `.ics` files and uses the following properties:  
- `SUMMARY`  
- `DTSTART`  
- `DTEND`  
- `DESCRIPTION`  

---

## Dependencies  
- `icalendar`  
- `PyQt5`  
- `tkinter`  
- `datetime`  

---

## How to run?  
Run the setup script(example.ics file needed): 
```bash
./setup.sh
